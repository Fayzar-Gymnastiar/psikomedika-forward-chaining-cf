## Instalasi

1. Clone repository

```bash
git clone https://github.com/USERNAME/psikomedika.git
```

2. Pindahkan ke folder htdocs.

3. Import database.sql ke MySQL.

4. Jalankan melalui XAMPP/Laragon.

5. Buka

```
http://localhost/psikomedika
```
